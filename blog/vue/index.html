初识Vue

2022年7月15日 · 阅读需 3 分钟

内容渲染​

html部分提供一个容器，{{}}获取数据

<div id="counter">
  Counter: {{ counter }}
</div>

js部分createApp创建Vue应用，mount方法把应用挂载到容器上。传递给 createApp 的选项用于配置根组件。当我们挂载应用时，该组件被用作渲染的起点。

const Counter = {
  data() {
    return {
      message: 'Hello Vue!!',
      counter: 0,
      isActive:true
    }
  }
}

Vue.createApp(Counter).mount('#counter')

可以使用v-bind命令设置标签属性，v-bind可省略

<div v-bind:title="message"><div>

鼠标停留显示返回内容

<div :class="isActive?'active':null"><div>

根据返回的数据中isActive的值设置类名

处理事件​

可以用v-on指令添加事件监听器，v-on:可简写为@

<div v-on:click="handleClick"></div>
const Counter = {
  data() {
    return {
      message: 'Hello Vue!!',
      counter: 0,
      isActive:true
    }
  },
  methods:{
    handleClick() {
        this.isActive = !this.isActive
    }
  }
}

methods配置方法，点击div改变类名

条件与循环​

v-model 指令能实现表单输入和应用状态之间的双向绑定

v-if指令能控制切换元素，根据表达式的值的真假来插入/移除元素

v-for 指令可以绑定数组的数据来渲染一个项目列表：

<div id="list-rendering">
  <ol>
    <li v-for="todo in todos">
      {{ todo.text }}
    </li>
  </ol>
</div>

根据数据中的todos循环渲染li

const ListRendering = {
  data() {
    return {
      todos: [
        { text: 'Learn JavaScript' },
        { text: 'Learn Vue' },
        { text: 'Build something awesome' }
      ]
    }
  }
}

组件化构建应用​

在字符串模板或单文件组件中定义组件时，定义组件名的方式有两种：使用 kebab-case (短横线分隔命名)，使用 PascalCase (首字母大写命名)

js部分

const TodoItem = {
  template: `<li>This is a todo</li>`
}

// 创建 Vue 应用
const app = Vue.createApp({
  components: {
    TodoItem // 注册一个新组件
  },
  ... // 组件的其它 property
})

// 挂载 Vue 应用
app.mount(...)

html部分

<ol>
  <!-- 创建一个 todo-item 组件实例 -->
  <todo-item></todo-item>
</ol>

关于props，能将数据从父组件传入子组件，用v-bind传入，组件的props属性接收，如:

<todo-item
      v-for="item in groceryList"
      v-bind:todo="item"
      v-bind:key="item.id"
    ></todo-item>

其中groceryLis来自父组件

const TodoItem = {
  props: ['todo'],
  template: `<li>{{ todo.text }}</li>`
}
